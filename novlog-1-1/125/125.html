#SOLVE

알파벳의 개수를 저장할 alpha[26] 배열을 선언하고, 문자열을 입력받아 아스키코드를 이용해서 해당하는 알파벳의 개수를 1씩 추가시켜 주었다.

정말 간단한 문제라서 다른 좋은 방식으로 풀이할 수 있나 고민해 보았는데,,, 아직은 잘 모르겠다.

아스키코드에 대한 개념만 알고 있었다면 간단하게 풀이할 수 있는 문제였다.

#CODE

#include <iostream>
#include <string>
using namespace std;
int alpha[26];

int main()
{
	string str;
	cin >> str;
	for (int i = 0; i < str.length(); ++i) {
		int temp = str[i] - 97;
		alpha[temp] += 1;
	}
	
	for (int i = 0; i < 26; ++i) {
		cout << alpha[i] << " ";
	}
	
	return 0;
}
