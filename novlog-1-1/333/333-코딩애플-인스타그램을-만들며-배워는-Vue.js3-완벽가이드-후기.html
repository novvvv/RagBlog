방학기간을 활용해 코딩애플 강사님의 Vue3.js 강의를 듣게 되었다.

더 적게 소요될 것이라고 생각한다.

 

 

# React 대신 Vue3.Js를 선택한 이유

한국에서 가장 인기있는 프론트앤드 프레임워크라고 하면 대부분 React를 뽑는다.

실제로 사람인 채용 사이트에서 React와 Vue를 검색해보면 대략 2배정도 차이 나는걸 확인할 수 있다.

 

 

 

그럼에도 불구하고 vue3.js 를 선택한 이유는

애당초 프론트앤드 직군 취업을 목표로 공부를 시작한게 아니었으며, 무엇보다 vue3.js는 쉽다!

백엔드 공부도 버거운 상황에 React 까지 파고들어 공부할 여력이 없었다.

vue는 약간의 Html, Css, Js 지식이 있다면 큰 러닝커브 없이 익힐 수 있다.

(물론 단순 찍먹 수준에서)

 

# 강의 장점

- 정말 vue3.js 라이브러리에 대한 지식이 전무하더라도 초심자의 눈높이에서 알기쉽게 설명해주신다.

- 강의당 시간이 10분 내외로 측정되어 있어 부담없이 강의를 시청할 수 있다.

- 별도의 Html, css 양식을 제공해주어 vue3.js 기능을 익히는데 온전히 집중할 수 있다.

 

# 강의 단점

장점이자 단점이 될 수 있다고 생각하는데 너무 초심자 눈높이에서 설명해 주시기에 확실히 강의의 깊이는 부족했다. 만약 프론트 직군으로 준비한다면 이 강의 만으로는 무리고 추가적인 학습이 필요할 것 같다.

그럼에도 vue3.js에 대한 기능을 빠르게 익히는데 큰 지장은 없었으며, 만약 백엔드 개발 직군을 준비하는데 간단한 프론트앤드 UI가 필요한 경우에는 해당 강의에서 나온 내용 만으로도 크게 무리 없이 프로젝트를 진행할 수 있다

고 생각한다.
