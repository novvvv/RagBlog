주어진 배열을 그대로 복사하여 뒤에 추가한 새로운 배열을 만들면 되는 간단한 문제이다.

C++ STL Vector 컨테이너에서 제공하는 insert 멤버함수를 사용해 문제를 풀이했다. → insert member function

 vector<int>::iterator it = nums.insert(nums.end(), nums.begin(), nums.end());

#CODE

class Solution {
public:
    vector<int> getConcatenation(vector<int>& nums) {
        vector<int>::iterator it = nums.insert(nums.end(), nums.begin(), nums.end());
        return nums;
    }
};
