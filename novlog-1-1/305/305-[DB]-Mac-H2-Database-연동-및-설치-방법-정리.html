위 주소로 이동하여, Platform-Independent Zip 파일을 다운로드 받는다.

 

 

윈도우는 별도의 권한 설정 없이도, h2 db 파일을 실행시킬 수 있지만,

맥은 따로 터미널에서 권한을 부여해 주어야 한다.

 

h2 데이터베이스 디렉터리의 bin 폴더로 이동한 뒤

 

cd h2 // h2 directory 이동
cd bin // bin directory 이동

 

h2.sh 파일에 대한 권한을 허용해 준다. 

 chmod는 Change Mode의 약자로, 유저에게 권한을 부여할 때 사용한다.

 

chmod 755 h2.sh

 

그러면 아래와 같은 웹페이지가 열릴텐데, 좌측 하단의 연결을 눌러주면,

h2 데이터베이스가 로컬에 생성된다.

 

 

콘솔로 돌아가 파일 리스트를 확인하여
