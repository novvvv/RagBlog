#1 액티비티(Activity)

어플리케이션을 제작할 때 하나의 화면 단위를 액티비티 (Activity) 라고 부르며, 앱 실행 시 처음 출력되는 화면을 메인 액티비티 (Main Activity) 라고 부른다.따라서 처음 프로젝트를 생성하면 자동으로 메인 액티비티가 생성되고, 메인 액티비티를 위한 자바 소스 파일인 MainActivity.java 파일이 생성된다. 그리고 생성된 메인 액티비티는 AndroidManifest.xml 파일에 등록이 된다.

#2 첫 화면 변경

 

그렇다면 첫 화면을 변경하는 방법은 간단하게 유추가 가능할 것 이다. 바로 메인 액티비티를 관리하는 AndroidManifest.xml 파일의 소스코드를 수정하면 된다. AndroidManifest.xml 파일은 app > mainfests 경로로 들어가면 확인 가능하다.

아래의 빨간색 네모 부분을 원하는 액티비티 파일 명으로 변경하면, 어플리케이션을 실행 시 처음으로 출력되는 액티비티가 변경되게 된다.
